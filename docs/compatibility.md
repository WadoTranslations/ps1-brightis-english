# Compatibility

Brightis is a normal, well-behaved PS1 game, and the patch does not change
that. It should run anywhere the original disc runs. The list below is where
it has actually been checked. If you play it somewhere else, a quick report
either way helps fill this in.

| Setup                     | Status           | Notes                                  |
|---------------------------|------------------|----------------------------------------|
| DuckStation               | Tested           | Primary emulator during development.   |
| PCSX-Redux                | Tested           | Works fine.                            |
| Beetle PSX / RetroArch    | Expected to work | No known issues; reports welcome.      |
| ePSXe                     | Expected to work | Older, less accurate; reports welcome. |
| Real hardware             | Not yet verified | Should behave like any backup; see below. |

## Settings

There is nothing special to configure. Default settings on an accurate
emulator are fine. If your emulator lets you pick a BIOS, use a Japanese
(SCPH-XXXX/J) BIOS to match the region of the disc.

## Real hardware

Playing a burned disc or running it from an ODE or flash cart should work
the same as any other PS1 backup on your setup; the patch does not touch
anything that would change that. Nobody has confirmed it yet though, so if
you try it, please report back either way.
