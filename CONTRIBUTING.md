# How this project works

Thanks for wanting to help. Here is what is useful and what is not.

## Please do send

- **Bug reports.** Typos, garbled or overflowing text, lines that run off the
  box, anything left in Japanese, a wrong speaker name, a crash, a softlock.
- **Compatibility reports.** "Runs fine on X" or "will not boot on Y" are
  both worth knowing.

A clear, specific report with a screenshot and a rough location in the game
is the most helpful thing you can do. I read all of them.

## Please do not send

- **Code or script pull requests.** The tools and the working script live in
  a separate private repo. There is nothing here to submit a patch against,
  and the translation itself is mine to curate, so it is closed to outside
  edits. If a line reads wrong to you, tell me in an issue and I will look.
- **Requests for the game disc**, or for the extraction and insertion tools.
  I cannot help you find the game, and the tools are not being released.

None of this is meant coldly. The repo is deliberately small so that bug
reports do not get lost. Thanks for playing.
